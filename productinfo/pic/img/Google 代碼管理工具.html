進階自訂功能：您可以在自訂 HTML 代碼中插入變數，只要以 {{variableName}} 包住變數名稱即可；如果您要在多個環境中重複使用代碼，這項功能就很實用。
將變數參照升級為 JavaScript 物件後，變數就可以在 HTML 欄位中當做一般的 JavaScript 物件使用。注意：如果您使用過去只看字串的方式來解讀變數，可能就必須先修改 JavaScript 程式碼，再進行升級。

某些代碼無法在非同步環境裡順利運作，而比較適合在「friendly iframe」裡運作。使用「document.write()」的代碼可能就屬於這一類。在使用此模式之前，請務必先與代碼供應商確認是否支援 friendly iframe。
